# 🛍️ PastelShop – Modern eCommerce Web Application

A fully responsive eCommerce web application built with React and Tailwind CSS, featuring a premium soft pastel UI and a complete checkout flow.

---

## 🚀 Live Features

- 🛒 Cart Page (Responsive 70% / 30% Layout)
- 🏠 Address Page (Shipping Information)
- 💳 Payment Page (Card / UPI / COD Selection)
- 🔄 Loading Spinner & Splash Screen
- ✅ Order Success Page with Redirect
- 📞 Contact Page
- 📘 About Page
- ❌ 404 Error Page
- 🌐 API Error Handling & Network Fallback
- 📱 Fully Responsive Design

---

## 🎨 UI Theme

Soft Pastel Premium Design  
Primary Color: `#7C3AED`  
Accent Color: `#F59E0B`  
Background: `#F3F0FF`

Designed for a modern, clean, and startup-level shopping experience.

---

## 🧰 Tech Stack

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🔀 React Router DOM
- 🎯 Lucide Icons
- 🧠 Functional Components & Hooks
